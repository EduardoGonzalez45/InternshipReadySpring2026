# 🤖 About aiBot.py:

### Model name: ✨ Google Gemini, Free Tier ✨ From: Google
Google Gemini was chosen because of its superior developer tools and its cost-effectiveness, and the free tier was chosen specifically for lightweight simple document summaries, so there isnt a need for many requests to be made.

### Reflection on responsible AI use:
I think that AI is responsibly used when you know what you are doing. I should know how to evaluate and control the outputs of an AI bot to my discretion to provide results that will contain little bias or misinformation, much like in this local and simple AI Bot API that I have constructed. AI is best used as a tool to support what I make, not as an end-all answer to any problem that I have. It is important for anyone who is planning to build an API like this or a bot to understand the ethical implications of using an AI Bot and make sure the bot is used ethically.
